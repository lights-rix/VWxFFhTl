# 项目简介

## 核心功能
- 基于YOLOv8的钢筋实例分割检测系统
- 支持图片、视频及摄像头进行检测
- 支持选择模型
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并在界面直接查看所有检测结果
- 支持检测结果保存
- 支持选择单张图片或图片文件夹进行识别
- 支持选择视频文件进行识别
- 支持调用本地摄像头进行识别
- 支持自定义置信度、IOU阈值
- 支持选择显示标签和原图
- 支持选择检测模型
- 包含1300张数据集，包含['rebar']，1类

## 适用场景
- 钢筋检测和实例分割任务
- 工程质量检测
- 建筑行业自动化检测
- 计算机视觉研究和应用

## 技术特点
- 基于Pycharm+Anaconda或VSCode+Anaconda开发
- 使用python3.9、opencv-python、PyQt5、ultralytics、torch1.9等环境
- 提供完整源码、完整标注的数据集、训练好的模型及训练结果
- 包含项目运行教程
- 配置成功环境可直接使用
- 提供UI界面源文件、图标等
